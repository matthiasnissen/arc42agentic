# Projekt-Konfiguration für arc42-Review-Agenten

## Dokumentationspfad

Der Pfad zur arc42-Architekturdokumentation innerhalb dieses Repositories:

```
arc-doc/
```

Alle arc42-Review-Agenten verwenden diesen Pfad als Wurzelverzeichnis für die Dokumentation. Die Sektionsordner (`01-Einfuehrung-und-Ziele/`, `02-Randbedingungen/`, etc.) werden relativ zu diesem Pfad erwartet.

**Anpassung:** Um eine andere Dokumentation zu reviewen, passe den Pfad oben an oder gib den Pfad direkt im Prompt an — eine explizite Angabe im Prompt hat Vorrang vor dieser Konfiguration.
